mnml Icons version 2.2

A set of minimal icons to help you iconify that feature ;) powered by @font-face

In order to use this font you must use the class .p-icon eg. <span class="p-icon">M</span> this will show a Star M

But a more semantic way to do this is calling the icon by name eg. <span class="p-star"></span> this will show a Star

Some of the mixins that are included

Font Mixin: .font();
  // expects: @weight, @size, @lineheight, @fontfamily
  // defaults: normal, 16px, 20px, @sans
  // note that the variable @sans can be changed in perkins.config.less

  // DEMO:
  .p-fontmixin {
    .font(bold, 28px, 32px, "Georgia", Times New Roman, serif);
    text-align: center;
  }

Radius Mixin: .radius();
  // expect: @radius
  // default: 5px
  // note that you can get fancy with it and selectively apply rounding to certain corners

  // DEMO:

    .radius(0 10px 5px 40px);
    border: 1px solid darken(@orange, 20%);
    background: @orange;
    padding: 10px;
    text-align: center;
    color: @white;

Gradient Mixin: #gradient > .v(); + #gradient > .h(); + #gradient > .hline();
  // .v() expects: @startColor, @endColor, @img
  // default: #555, #333, none
  // .h() expects: @startColor, @endColor, @img
  // default: #555, #333, none
  // .hline() expects: @startColor, @midColor1, @midColor2, @endColor, @img
  // default: #B0BAC5, #9DA7B2, #919CA8, #7E8A98, none

  // DEMO:
  .p-gradientV {
    #gradient > .v();
